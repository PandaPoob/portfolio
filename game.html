Whack-A-Panda

Mockup

Introduction

Tech stack: HTML, CSS & JavaScript

During this project I created my very own game. The game is a simple clicking-game inspired by the classic arcade game Whack-A-Mole. This was my first time working with JavaScript. Throughout the project I learned the basics and the logic of the language. I was also introduced to thinking about ways to engage and motivate users with interactive animations.

Idea

The theme of the game stems from the name I use online "PandaPoob". That is why the main elements are pandas and poops. I wanted to retain the feel of the game I was inspired by (Whack-A-Mole) and therefore I created my backgrounds as if you were playing on an arcade machine. To strengthen this old school feel I created all of my art elements as pixel art. Everything was created in Adobe Illustrator.

Process

Since this was my first experience with JavaScript I started small to not overwhelm myself. First I planned and made a state machine diagram to get an overview of my game. Then coded the animations and functions and tested them to make sure they worked. During this phase I got very familiar with the Firefox inspector and console. After finishing the main animations and functions I carefully linked it all together.
After the game was working I tested it on users and adjusted based on the feedback. I slowed down the speed because the majority of users thought the game was too hard and I also added text to my info screen to clarify the rules.
