Hacked Hogwarts Student List

Mockup

Introduction

Tech stack: HTML, CSS, JavaScript & JSON

In this project I pretended to be a newly hired frontender for the wizard school Hogwarts. I built a student list from the given data and also made the navigation easier by adding functions like filtering, searching etc. I also needed to able to appoint different students to different squads or roles and even expel them. Furthermore you can also click on each individual student card to see more details. If you are feeling rebellious you may want to consider clicking the button at the bottom of the page....

Process

Because this assignment had a lot of requirements, some even conflicting, I had to carefully plan my course of action. I did so by first making a list of all the functions I knew I needed and then breaking down the different functions into smaller activity diagrams that helped me maintain an overview. I started small with the requirement of wanting to show a list of students. I did this by fetching the data, cleaning it, creating my own object called Student with the cleaned data, cloning them and finally displaying them in the html. Afterwards I added other features function by function. Since this was a very complex assignment I had to use a lot of return values and parameters. I also had to use closure as some of my functions worked inside of my details popup.

See full activity diagram here!

UX

To make the student list more visually interesting and clear I created small icons using Adobe Illustrator. I made house crests for my filter, icons for active or expelled students and for the inquisitor squad and prefects. I also did this to make the whole student list more coherent. I also added a background color to the filter that is chosen to let the user know what filter is currently active. The colors showcased are also linked to the students' house. For example all the Gryffindor students have a red heading on their card and a red background in their details popup. I did this through JavaScript.
